Introduction FoxAuth est un TOTP client de navigateur NON OFFICIEL qui est capable de saisir automatiquement les codes TOTP avec le support de FireFox et FireFox Android Caractéristiques
  • Support de Firefox Android
  • QR code / ajouter manuellement les informations de compte
  • Un support de Onglet Conteneur pour saisir automatiquement plusieurs comptes du même domaine
    • importer / exporter
      • importation depuis d'autres authentifiants (en cours)
  • Liste de comptes (basée sur un mot de passe) encryptée
  • Saisir automatiquement le formulaire de TOTP
  • Dropbox / (dans les prochaines versions) FxA sync
  • Photon Design
Avertissements Pendant la sync, vous êtes PAS censé avoir l'un des comportements énumérés ou vous êtes totalement responsable de la perte de vos données:
  1. Déplacement, renommage, suppression ou édition manuelle de tout fichier de votre dossier d'application (le copier-coller ne doit toutefois pas causer de problèmes).
  2. exploiter les informations de compte entre plusieurs appareils dans une fréquence VRAIEMENT haute.
  3. utiliser des mots de passe de cryptage différents selon les appareils.
Vous devriez également noter que si vous connectez un Firefox avec une base de données FoxAuth encrtypée, toutes les bases de données non encryptées de Firefox FoxAuth seront remplacées. Sécurité Nous utilisons le code de cryptage de Firefox Send dans l'API WebCrypto. Bienvenue pour tout examen de sécurité, il sera plus efficace.
